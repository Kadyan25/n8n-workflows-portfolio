# Website Checker (n8n)

## Goal
Evaluate a website URL with AI, generate actionable feedback, and publish output as a tracked artifact.

## Included workflow

1. **Website Checker Pipeline**
   - Source: `Website Checker.json`
   - Fetches website content, runs multi-step AI analysis, and transforms results into structured output.
   - Produces final report content and creates a file in GitHub for traceability.

## Demonstrated skills
- Automated website auditing workflow design
- Multi-agent AI analysis and synthesis in n8n
- HTTP content retrieval and transformation with code nodes
- GitHub file automation for report storage/versioning
- Reusable quality-check pipeline for web operations
